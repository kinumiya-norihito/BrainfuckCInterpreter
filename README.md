# BrainfuckCInterpreter
BrainFuckのインタプリタ。
C言語で書かれている。
このソースコードをコンパイルして使う。

第1引数の文字列をBrainFuckのファイル名として解釈し、対象のファイルの実行する。
第1引数の文字列を`-a`にすると第2引数をBrainFuckのソースコードとして解釈して実行する。

以下はターミナルでの記述例

コンパイル
```
 $ cc BrainFuck.c -o bfci
```

ファイルを実行
```
 $ ./bfci helloworld.bf
```

ファイルを使わず実行
```
 $ ./bfci -a "++++++[->++++[->+++<]<]>>.<+++++++[->++++<]>+.+++++++..+++.>++++[->++++[-<<---->>]<]<---. [->++<]>-.<++++++[->++++<]>.+++.------.--------.<<++++[->++++[->----<]<]>>---...[-]++++++++++."
```
